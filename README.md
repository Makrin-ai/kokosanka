Эмулятор командной строки 

Проект представляет собой эмулятор интерфейса командной строки (Shell REPL) для UNIX‑подобных систем, написанный на Java. Эмулятор имитирует базовое взаимодействие с shell, включая поддержку переменных окружения, исполнение скриптов и настраиваемые параметры запуска.

Основные возможности
-   REPL (Read‑Eval‑Print Loop)**: Интерактивный режим с приглашением вида `username@hostname:~$`.
-   Парсинг команд**: Поддержка раскрытия переменных окружения (например, `$HOME`).
-   Базовые команды‑заглушки**: `ls`, `cd`, `exit`.
-   Конфигурация через параметры командной строки**: Указание путей к VFS и стартовому скрипту.
-   Исполнение скриптов**: Последовательное выполнение команд из файла с отображением ввода и вывода.
-   Обработка ошибок**: Информирование об ошибках ввода, неизвестных командах и проблемах со скриптами.

Требования:
-   Java Development Kit (JDK) 8** или выше.
-   Операционная система**: Любая, поддерживающая Java (Linux, macOS, Windows). Для полноценной имитации рекомендуется UNIX‑подобная среда.

Код программы расположен по следующему пути: src/Main
Тестовые скрипты расположены по следующему пути: lib/test1.sh || lib/test2.sh

Сборка и запуск:

1.  Клонируйте репозиторий**:
    ```bash
    git clone <URL-вашего-репозитория>
    cd <имя-папки-проекта>
    ```

2.  Скомпилируйте проект**:
    ```bash
    javac Main.java
    ```

3.  Запустите эмулятор**:

    *   В интерактивном режиме (REPL)**:
        ```bash
        java Main
        ```

    *   С параметрами конфигурации**:
        ```bash
        java Main --vfs /путь/к/vfs --script /путь/к/скрипту.sh
        ```
